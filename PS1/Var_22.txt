1. Проверить, является ли билет счастливым по-московски (abcdef => a+b+c = d+e+f)
2. Даны три натуральных числа f, s и n. По первому члену арифметической прогрессии f и шагу прогрессии s определить каким по порядковый номер числа n в прогрессии (или вывести -1, если число не принадлежит ей)
3. Найти произведение цифр в k-ичной системе счисления (k от 2 до 10) десятичного натурального числа n (n<=10^7)
4. На вход подаётся последовательность из n целых чисел (по модулю <=10^9). Определить является ли последовательность строго убывающей (каждый элемент < предыдущего)
5. Разложить чётное целое число на сумму двух простых чисел. Нужно вывести пару с наименьшим простым числом
6. Найти последнюю цифру N! (N<10000)
