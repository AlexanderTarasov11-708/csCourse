1. Не используя других массивов, переставить элементы целочисленного массива в обратном порядке.
2. Значения массива сдвинуть циклически вправо на одну позицию так, чтобы последний элемент стал первым.
3. Значения массива сдвинуть циклически вправо на k позиций.
4. Двумерный массив размера n x n содержит только положительные значения. Замените на ноль значения всех элементов, расположенных на главной и на побочной диагоналях. Главная диагональ идет из верхнего левого угла в правый нижний угол, побочная диагональ идет из верхнего правого в левый нижний угол.
5. Двумерный массив размера n x n содержит не равные нулю значения. Замените на ноль значения всех элементов, расположенные выше побочной диагонали.
