1. Проверить, является ли билет счастливым по-питерски (abcdef=>a+c+e=b+d+f)
2. Возвести вещественное число n в целую степень k (используя алгоритм быстрого возведения в степень – бинарный/справа налево)
3. По заданному числу k, вывести символ 0 столько раз, сколько он встречается в двоичном представлении числа
4. На вход подаётся последовательность из n целых чисел (по модулю <=10^9). Посчитать число перемен знака в последовательности.
5. Вычислить, какое минимальное число квадратов натуральных чисел требуется для представления заданного числа n. Примеры: 7=1*1+1*1+1*1+2*2, а 13=2*2+3*3
6. По заданному n (от 0 до 100) найти сумму всех n-значных чисел.
